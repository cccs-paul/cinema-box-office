## Capture Guidelines

1. **Resolution**: Capture at 1920x1080 or higher
2. **Format**: PNG format preferred for clarity
3. **Demo Data**: Use the Demo RC with sample data for realistic screenshots
4. **Login**: Use the demo user (admin/Admin@123)
5. **Browser**: Use Chrome or Firefox without extensions visible
6. **Clean State**: Ensure no error messages or loading spinners are visible

## Updating Screenshots

When the UI changes:
1. Delete the old screenshot
2. Capture the new screenshot with the same filename
3. Commit and push the changes
